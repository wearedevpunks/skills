# Bitbucket

Use Bitbucket Downloads, the Downloads API, or the Atlassian `bitbucket-upload-file` pipe. No first-party generic CLI assumed.

Refs:

- https://support.atlassian.com/bitbucket-cloud/docs/deploy-build-artifacts-to-bitbucket-downloads/
