# GitLab

Use GitLab package upload for neutral asset hosting. Use release upload when release UX matters.

- Package upload: `glab packages upload <file> --name <package> --version <version>`
- Release upload: `glab release upload <tag> [<files>...] --use-package-registry`

Refs:

- https://docs.gitlab.com/cli/packages/upload/
- https://docs.gitlab.com/cli/release/upload/
