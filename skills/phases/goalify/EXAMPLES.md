# Goalify Examples

Use these as internal shapes only. Do not print them as ready-to-paste output.

## Delivery Shape

`/goal Deliver <outcome> in <repo/path>. Read <sources>. Stay in <scope>. Validate with <commands>. Done when <evidence>. Stop if <blocker>.`

## Debug Shape

`/goal Debug <failure> in <repo/path>. Reproduce with <scenario>. Patch only proven cause. Validate with <commands/logs>. Done when <scenario passes>. Stop if <external blocker>.`
